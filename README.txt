项目目的：
    1：提供番茄计时
    2：记录工作、学习内容，并提统计数据

具体功能
番茄计时：
    1：25-5min模式的番茄计时，并有可自定义时间长度功能
    2：设定目标的预计番茄数
    3：统计数据输出，包括按天、按周、按月的统计数据

内容记录：
内容记录主要分为两块：工作，学习。
记录工作的目的是为了知道自己在工作中的成长，并且为工作中的周报、月报等提供基础数据，主要包括：
    1：按天记录工作内容；
    2：按周\月输出总结，并且用户可编辑，保存为周\月报，突出做了什么，解决了什么难题，计划做什么；
记录学习的目的在于督促自己保持学习，应该包含以下内容：
    1：单项学习目标，结合番茄
    2：统计数据
